ImpDip Graph Game

Hello! Welcome the the node graph game. A few things to know before starting:

  • The Pacific nodes and the Chukchi Sea node have been removed so that the graph doesn't cross over itself. This can be reenabled in the options
  • This graph specifically covers the 2025 June-July version.
  • Each node represents either a sea province, a land province, OR A COAST
  • Coasts are not connected to their land province (e.g. Rome EC is not connected to Rome)
  • There are 796 nodes

This game is inspired by Ottus's Diplomacy Graph Puzzle, which is in turn inspired by Alcas' World Graph Puzzle.
